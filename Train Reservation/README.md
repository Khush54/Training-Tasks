# Train Reservation Management System

## Objective
To build a Train Reservation system that demonstrates the use of JavaScript array methods and DOM manipulation in a practical scenario.

## Technologies Used
- HTML
- CSS
- JavaScript

## Features
- Display Passenger List
- Show Senior Citizens (Age 60+)
- Convert Passenger Names to Uppercase
- Calculate Total Age using reduce()
- Calculate Total Age using reduceRight()
- Check if All Passengers are Adults
- Check if Any Passenger is a Minor
- Convert Seat Data into String Format
- Add Extra Passengers
- Separate First Passenger from the List

## Project Description
This project stores passenger details in a JavaScript array of objects.  
Each button performs a specific operation using different JavaScript array methods.

When a user clicks a button:
- The respective array method executes
- The result is generated
- The output is displayed dynamically in the result box using DOM manipulation

## What I Learned
- Working with arrays of objects
- Using advanced array methods
- DOM manipulation using innerHTML
- Template literals
- Dynamic UI updates
- Structuring JavaScript logic properly
