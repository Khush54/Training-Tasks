# School Result Analyzer

## Objective
To analyze student names and marks using JavaScript array methods and display the results dynamically on a web page.

## Technologies Used
- HTML
- JavaScript (Array Methods & DOM Manipulation)

## Concepts Covered
- indexOf()
- lastIndexOf()
- includes()
- find()
- findIndex()
- findLast()
- findLastIndex()
- DOM Manipulation using getElementById()

## Features
- Finds the first occurrence of a student name
- Finds the last occurrence of a student name
- Checks whether a student is enrolled
- Finds the first student scoring above 40 marks
- Displays the index of the first passing student
- Finds the last student scoring above 50 marks
- Displays the index of the last passing student

## What I Learned
- How to use modern JavaScript array methods
- Difference between indexOf() and findIndex()
- How find() and findLast() work
- Connecting JavaScript logic with HTML using DOM
- Displaying dynamic results inside a webpage
